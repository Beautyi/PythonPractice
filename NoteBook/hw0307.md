# 列表、元组、字典的异同
## 一、列表
### （一）定义
由一系列特定顺序排列的元素组成。
eg:
```python
names = ['Martin', 'Jack', 'Rose']
```
### （二）增删改查方法
#### 1、增加元素
##### ①在列表末尾增加元素
**append函数**（*面向对象的函数*）
##### ②在列表中插入元素
**insert(索引，新元素)函数**（*面向对象的函数*）
eg:
```python
insert(0,'Ava')#将‘Ava’添加至列表第一个元素的位置
```
#### 2、删除元素
##### ①用del语句删除元素
**del 列表名[索引]**
eg:
```python
del names[0]#删除列表中的第一个元素
```
+ del可以删除任何知道索引的列表元素
+ 删除后无法再访问
##### ②使用方法pop删除元素
**pop函数**（*面向对象的函数*）
+ 删除列表末尾的元素
+ 相当于弹出栈顶元素
##### ③弹出列表中任何位置处的元素
**pop函数**（*面向对象的函数*）
eg:
```python
names.pop(1)#删除第二个元素
```
+ 删除后可继续使用元素
##### ④根据值删除元素
**remove函数**
列表.remove(元素)
eg:
```python
names.remove('Martin')#删除'Martin'
```
+ 需要知道删除元素的值
+ 删除后可继续使用
+ remove只删除第一个指定的值，如列表中重复出现需用循环来判断。
#### 3、访问列表
**列表[索引]**
eg:
```python
print(names[0])#打印第一个元素
```
+ 索引从0开始，不是1
+ 索引为-1访问最后一个元素
#### 4、修改列表元素
eg:
```python
names[0] = 'Ava'#将第一个元素修改为‘Ava’
```

## 二、元组
###（一）定义
Python将不能修改的值称为**不可变的**，而不可变的列表被称为**元组**。
+ 元组看起来犹如列表，但使用*圆括号*而不是方括号来标识。
eg:
```python
dimensions = (200, 50)
```
+ 不可以修改，但可以给元组重新赋值。
----
****
## 三、字典
###（一）定义
~~Python~~**字典**是另一种可变容器模型，且可存储任意类型对象，如字符串、数字、元组等其他容器模型。
eg:
```python
dict = {'Alice': '2341', 'Beth': '9102', 'Cecil': '3258'}
```
#### 1.访问字典中的值
eg:
```python
print "dict['Name']: ", dict['Name'];
print "dict['Age']: ", dict['Age'];
```
#### 2.修改字典中的值
eg:
```python
dict['Age'] = 8
```
#### 3.添加键-值对
eg:
```python
dict['School'] = "DPS School"
```
#### 4.删除键-值对
eg:
```python
del dict['Name']; # 删除键是'Name'的条目
dict.clear();     # 清空词典所有条目
del dict ;        # 删除词典
```
+ 删除后不可再使用
